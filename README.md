# zazcar-frontend

node version 4.4.7
Precisa ter instalado globalmente - gulp, bower, RubyGems e Gem bundler

Na pasta do projeto:
1 - npm install
2 - bower install
3 - bundle install

Para rodar o projeto:
gulp server

Para gerar o build:
gulp build


Funcionando no Amazon S3
http://dribbble-shots-teste.s3-website-sa-east-1.amazonaws.com
